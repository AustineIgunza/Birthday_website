# 🎂 Happy Birthday Website

A beautiful, interactive birthday website for Imani! Built with HTML5, CSS3, and Vanilla JavaScript.

## 📋 Features

✨ **5 Interactive Pages:**
1. **Welcome Page** - Greeting with animated buttons
2. **Gifts Page** - Three gift cards (Love Letter, Photos, Surprise)
3. **Love Letter** - Heartfelt message from hubby
4. **Photo Gallery** - Beautiful moments shared together
5. **Surprise** - Dinner reservation reveal

🎨 **Design Elements:**
- Responsive design (works on all devices)
- Smooth animations and transitions
- Interactive elements (clickable hearts)
- Mobile-optimized for phones
- Beautiful gradient backgrounds
- Floating decorative emojis

📱 **Mobile Friendly:**
- Optimized for phones (360px - 480px)
- Tablets (768px+)
- Desktop (1920px+)
- Touch-friendly buttons and interactions

## 🚀 Deployment

Deployed live on **Netlify** - anyone with the link can access it!

### How to Deploy Changes:
1. Edit `index.html` in VS Code
2. Save the file (Ctrl+S)
3. Run `deploy.bat` to push to GitHub
4. Netlify auto-deploys (1-2 minutes)
5. ✅ Changes go live!
